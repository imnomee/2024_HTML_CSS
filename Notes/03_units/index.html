Pixel Units in CSS

Pixel Units are absolute units, which depends on the dot of the screen

Percent Units in CSS

Percent Unit are Relative Units, Depends on the parent element

EM Units in CSS

EM Unit are also Relative Units, Depends on the parent element or the browser base font-size (16px === 1em)

REM Units in CSS

REM Unit are also Relative Units, Depends on the root element (like html font size) or default base font-size (16px === 1em)

VH - VW Units in CSS

VH - VW Unit are percent of the screent.
95vh === 95% height of the view screen
90vw === 90% width of the view screen
